# Recommender System based on Whale Optimization Algorithm (WOA)

This project implements a recommendation system using the **Whale Optimization Algorithm (WOA)** for hyperparameter optimization and model training in Matrix Factorization (MF).

## Project Structure
- `src/`: Core implementation of MF, WOA, PSO, and evaluation metrics.
- `notebooks/`: Jupyter notebooks for training, comparison, and visualization.
- `models/`: Best parameters and trained model weights.
- `results/`: Performance metrics and search results.
- `figures/`: Visualizations of convergence and performance.
- `data/`: Dataset storage (MovieLens 1M).
